# User Assistant – Frontend Application

این پروژه بخش فرانت‌اند «دستیار کاربر پلتفرم تحلیل داده» است که با استفاده از JavaScript و Electron پیاده‌سازی شده است.  
هدف این برنامه، راهنمایی مرحله‌به‌مرحله کاربران از طریق ساختار FAQ و در صورت نیاز، هدایت آن‌ها به مسیر ثبت تیکت پشتیبانی است.


---

## تکنولوژی‌های استفاده‌شده

- JavaScript
- HTML / CSS
- Electron
- Node.js
- npm

---

## پیش‌نیازهای اجرا

برای اجرای پروژه، نصب **Node.js** الزامی است.

دانلود از:
https://nodejs.org  
(نسخه LTS پیشنهاد می‌شود)

---

## نحوه اجرای پروژه

پس از دریافت فایل‌های پروژه:

1. وارد پوشه پروژه شوید
2. در  Command Prompt دستور زیر را اجرا کنید:
**npm install**
این دستور وابستگی‌ها را نصب کرده و پوشه `node_modules` را ایجاد می‌کند.

3. سپس برای اجرای برنامه:
**npm start**

## اجرای سریع در ویندوز

در سیستم‌های ویندوز، می‌توان برنامه را با دوبار کلیک روی فایل زیر اجرا کرد:

run.bat

این فایل به‌صورت خودکار دستورات زیر را اجرا می‌کند:
- npm install
- npm start


---

## ساختار فایل‌ها

├─ index.html رابط کاربری برنامه
├─ app.css استایل‌ها
├─ render.js منطق فرانت‌اند
├─ main.js فایل اصلی Electron
├─ package.json تنظیمات پروژه و اسکریپت‌ها
├─ package-lock.json قفل نسخه وابستگی‌ها
├─ preload.js  واسط امن بین Renderer و Main
├─ run.bat اجرای سریع برنامه در ویندوز

---

## نکات مهم

- پوشه `node_modules` در پروژه قرار داده نشده و پس از اجرای `npm install` ساخته می‌شود.
- قبل از اجرای پروژه، از نصب صحیح Node.js اطمینان حاصل کنید.
- این پروژه در فاز فعلی فاقد هوش مصنوعی بوده و بر پایه ساختار FAQ طراحی شده است.
